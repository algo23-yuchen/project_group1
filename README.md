# 2. “代码”文件夹包含我们需要的所有代码实现和数据。里面有三个文件夹，分别是：
  “数据库”文件夹，其中包含从数据库中读取数据并在其中写入数据的代码。   
  “交易者”文件夹包含我们生成信号的策略、回测和算法交易系统的主要组件。   
  “data”文件夹包含生成信号和回测策略所需的数据。   
  “结果”文件夹包含我们所有的回测结果以及图片。   
  “5210project - Group1（第4次演讲）.pptx”是我们演讲的幻灯片。

# 2. 文件说明
  “database”文件夹，其中包含从数据库中读取数据并在其中写入数据的代码。   
  “Trade”文件夹包含我们对交易的定义，买入或者卖出，以及数量。“OrderBook.py”完成了订单簿数据获取和撮合市价订单的部分，“twap.py”完成了分批下单的部分。
  “strategy”文件夹包含了主要策略和指标评价构建
  “data”文件夹包含生成项目所需的数据。      
  “5210project - Group1.pptx”为演示ppt。
